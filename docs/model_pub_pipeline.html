为什么可以复用？

🔗 共享层 (全局唯一，所有模型复用)
dat_pbs → depth_pub → trade_flow_feature_pub
▼
🔀 分叉层 (每个模型独立实例)
fusion_factor_pub(model_X) → model_pub(model_X)
  • 1IPC 路径设计：上游频道不带 model_name（如 factor_pub/{venue}/trade_flow_feature），是公共路径，全局只需一套
  • 2iceoryx2 多订阅者：支持 max_subscribers=10，多个 fusion_factor_pub 实例零拷贝读取同一份共享内存数据，无额外开销
  • 3分叉点在 fusion_factor_pub：输出路径带 model_name（fusion_factor/{venue}/{model_name}），不同模型从这里各走各的
  • 4新增模型零侵入：只需部署一对 fusion_factor_pub + model_pub，上游 dat_pbs / depth_pub / trade_flow_feature_pub 完全不动，不增加交易所连接数
